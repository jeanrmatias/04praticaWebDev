# JavaScript na Prática — 5 Exemplos com JS Puro

Repositório com 5 práticas de JavaScript puro, cada uma em sua própria pasta, baseadas exclusivamente no conteúdo dos slides `03_JavaScript.pdf` e `04_JavaScript-Pratica.pdf`.

## Estrutura

1. `pratica01-adicionando-conteudo/` — DOM (innerHTML, createElement, appendChild, removeChild) + Event Listener
2. `pratica02-troca-classe-cor-texto/` — DOM (classList.toggle, style, innerText) + Event Listener
3. `pratica03-leitura-forms-calcula/` — DOM (value, innerHTML, insertRow) + Event Listener (submit) + OOP (Math.pow)
4. `pratica04-adiciona-elementos/` — DOM (createElement, appendChild, removeChild, classList) + Event Listener + OOP (propriedades)
5. `pratica05-capturando-localizacao/` — BOM (navigator.geolocation) + Event Listener + JS client-side

## Tecnologias

HTML5, CSS3 (mínimo), JavaScript puro

## Como usar

Abra o arquivo `index.html` de cada pasta diretamente no navegador.

## Práticas

① **Prática 01 - Adicionando Conteúdo HTML**

**Conceitos dos slides abordados:** DOM (alteração e adição/remoção de elementos HTML) e Event Listener.

**Métodos JS demonstrados:** `innerHTML`, `createElement`, `appendChild`, `removeChild`.

**Como interagir:** Clique nos botões para adicionar cards, parágrafos ou remover o último elemento.

② **Prática 02 - Troca de Classe, Cor e Texto de Elemento**

**Conceitos dos slides abordados:** DOM (classes, estilos CSS, texto de elementos) e Event Listener.

**Métodos JS demonstrados:** `classList.toggle`, `style.backgroundColor`, `innerText`.

**Como interagir:** Clique nos botões para alternar classe, mudar cor, alterar texto ou resetar o elemento.

③ **Prática 03 - Leitura de Forms e Cálculo**

**Conceitos dos slides abordados:** DOM (leitura de formulários, manipulação de tabelas), Event Listener (submit) e OOP (Math.pow).

**Métodos JS demonstrados:** `value`, `innerHTML`, `insertRow`.

**Como interagir:** Preencha nome, peso e altura para calcular o IMC e visualizar o resultado com classificação.

④ **Prática 04 - Adiciona Novos Elementos com JS**

**Conceitos dos slides abordados:** DOM (criação, adição e remoção de elementos), Event Listener e OOP (propriedades de objetos).

**Métodos JS demonstrados:** `createElement`, `appendChild`, `removeChild`, `classList`.

**Como interagir:** Adicione tarefas, marque como concluídas, remova individualmente ou todas de uma vez.

⑤ **Prática 05 - Capturando Localização**

**Conceitos dos slides abordados:** BOM (navigator.geolocation), Event Listener e JavaScript client-side.

**Métodos JS demonstrados:** `navigator.geolocation.getCurrentPosition()`.

**Como interagir:** Clique no botão para obter a localização atual e veja latitude, longitude e precisão.
